# SBA308-A
Description:
This project aims to provide a simple API for users to log their mood using emojis. Users can express their emotions for the day by selecting from a predefined list of emojis representing various moods. The project utilizes JavaScript for server-side and client-side functionality, allowing users to interact with the API seamlessly.

Features:

Log your mood for the day using emojis.
Choose from a diverse range of emojis representing different emotions.
Retrieve logged moods through API endpoints.
View statistics and trends of mood logs over time.
Secure authentication and authorization mechanisms for user privacy.
Technologies Used:

JavaScript (Node.js) for backend development.
Express.js for building the RESTful API.
MongoDB for storing mood log data.
HTML, CSS, and JavaScript for the frontend interface.
JSON Web Tokens (JWT) for user authentication.
npm packages for additional functionalities.
How to Run:

Clone the repository to your local machine.
Install dependencies using npm: npm install.
Set up a MongoDB database and configure the connection in the project.
Run the server: npm start.
Access the project through your web browser at the specified URL.
